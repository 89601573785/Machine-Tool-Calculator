# 🏭 Онлайн-сборщик производственного цеха

Веб-приложение для планирования и визуализации компоновки производственного цеха. Позволяет расставлять оборудование на рабочей области, связывать станки в технологическую цепочку и экспортировать результаты в PDF.

## ✨ Возможности

- **Каталог оборудования** — поиск и перетаскивание станков на рабочую область
- **Связи между станками** — Shift + клик по двум станкам создаёт линию соединения
- **Масштабирование и навигация** — зум (Ctrl + колесо мыши), перемещение (Alt/средняя кнопка мыши)
- **Сетка и привязка** — выравнивание оборудования по сетке
- **Сохранение и загрузка** — проекты в localStorage браузера
- **Экспорт/импорт** — обмен проектами в формате JSON
- **Расчёт и отчёт** — расчёт параметров и выгрузка в PDF

## 🛠 Стек технологий

| Компонент | Технология |
|-----------|------------|
| Интерфейс | HTML5, CSS3, JavaScript (ES6+) |
| Иконки | Font Awesome 6 |
| PDF | jsPDF, html2canvas |
| Хранение данных | SQL.js (SQLite в браузере) / JSON fallback |

## 📁 Структура проекта

```
server popa/
├── index.html          # Главная страница
├── css/
│   └── style.css       # Стили
├── js/
│   ├── main.js         # Основная логика (FactoryDesigner)
│   ├── connections.js  # Управление связями между станками
│   └── equipment-data.js # Каталог оборудования (fallback)
├── data/
│   └── factory.db      # База SQLite с данными оборудования
└── images/             # Изображения станков (.webp)
```

## 🚀 Запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/server-popa.git
   cd server-popa
   ```

2. Откройте `index.html` в браузере или запустите локальный сервер (рекомендуется для работы с SQLite):
   ```bash
   npx serve .
   # или: python -m http.server 8000
   ```

3. Откройте в браузере `http://localhost:3000` (или указанный порт).

> При открытии через `file://` SQLite может не загружаться — в этом случае используется встроенный каталог из `equipment-data.js`.

## ⌨️ Управление

| Действие | Клавиши/мышь |
|----------|---------------|
| Добавить станок | Перетащить из каталога на область |
| Связать станки | Shift + клик по первому, Shift + клик по второму |
| Зум | Ctrl + колесо мыши или кнопки +/− |
| Перемещение по области | Alt + левая кнопка / средняя кнопка / правая кнопка мыши |
| Удалить станок | Кнопка × на карточке станка |

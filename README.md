# Онлайн-сборщик производственного цеха

**Тема дипломного проекта:** разработка веб-решения создания и конфигурирования производственной линии ООО «ЛЕСКОМ».

Веб-приложение для планирования и визуализации компоновки производственного цеха: каталог станков и комплексов, рабочая область, технологические связи с выбором конвейера, расчёт линии и PDF-отчёт.

## Возможности

- Каталог оборудования с вкладками (все / станки / конвейеры / комплексы), поиском и drag-and-drop
- Развёртывание **комплексов** на поле с готовыми связями
- Связи между станками с выбором конвейера (мощность, габариты, скорость, цена)
- Режим **«Связать»** и быстрый способ через **Shift + клик**
- Адаптивная сетка, привязка к сетке, зум и панорамирование большой рабочей области
- Черновик в `localStorage` (ключ per user/project)
- Меню **«Проект»**: загрузка черновика, экспорт и импорт JSON
- Расчёт производительности линии (min по цепочке), энергии, стоимости, конвейеров
- Компактный PDF-отчёт с кириллицей
- **Интеграция с ЛК:** embed в iframe, сохранение через API, `postMessage` родителю

## Запуск

```bash
git clone https://github.com/89601573785/Machine-Tool-Calculator.git
cd Machine-Tool-Calculator
npx serve .
```

Откройте `http://localhost:3000` (порт может отличаться).

> **Важно:** при открытии через `file://` SQLite и API ЛК недоступны — используется `equipment-data.js`. Для полного функционала нужен HTTP-сервер (`npx serve .` или Live Server).

### Режим embed (как на сайте)

```
/?embed=1&projectId=<uuid>
```

Сохранение проекта в личный кабинет — через `integration.js` и `POST /api/v1/configurator/projects` с `credentials: 'include'`.

Подробно: [docs/CONFIGURATOR-INTEGRATION.md](docs/CONFIGURATOR-INTEGRATION.md)

## Верхняя панель

| Кнопка | Назначение |
|--------|------------|
| **Каталог** | Показать / скрыть боковую панель с оборудованием |
| **Очистить** | Удалить все станки и связи с поля |
| **Сохранить** | Сохранить черновик в браузере (`localStorage`) |
| **Проект** | Меню: **Загрузить** (черновик), **Экспорт** (JSON), **Импорт** (JSON) |
| **Связать** | Режим связей: клик по источнику, затем по приёмнику |
| **Как пользоваться** | Инструкция с описанием всех кнопок |
| **Сетка** | Включить / выключить сетку на поле |
| **Привязка** | Привязка станков к сетке (удерживайте **Ctrl**, чтобы временно отключить) |
| **− / + / ⊙** | Масштаб и центрирование рабочей области |
| **Рассчитать** | Расчёт линии; просмотр результатов или скачивание PDF |

## Управление

| Действие | Клавиши / мышь |
|----------|----------------|
| Добавить станок | Перетащить из каталога |
| Добавить комплекс | Вкладка «Комплексы» → перетащить на поле |
| Связать станки | **Связать** → источник → приёмник, или **Shift** + клик по двум станкам |
| Выбор конвейера | В модальном окне после создания связи |
| Редактировать связь | **ПКМ** по линии или подписи конвейера |
| Переместить станок | Перетаскивание карточки |
| Удалить станок | **×** на карточке |
| Зум | **Ctrl** + колесо или кнопки **±** |
| Перемещение по полю | **Alt** / СКМ / ПКМ + drag, или ЛКМ по пустому месту |

## Интеграция с leskomphp

Статика деплоится в `/configurator/` на том же домене, что и сайт (cookie сессии). API: `GET/POST /api/v1/configurator/projects`.

Данные в каталоге **учебные**, не официальный прайс ООО «ЛЕСКОМ».
